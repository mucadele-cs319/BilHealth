Namespace BilHealth.Services.AccessControl

Classes

AccessControlService

AccessControlService.NoAccessStrategy

Will not allow access under any circumstances.

AccessControlService.RelationalAccessStrategy

Only grants access if user has an active relation to the accessed entity. For example, an open case in which they are currently assigned.

AccessControlService.TimedAccessStrategy

Only grants access while the user has an unexpired TimedAccessGrant.

AccessControlService.TrackedAccessStrategy

Always grants access, however, all access is tracked in an AuditTrail.
