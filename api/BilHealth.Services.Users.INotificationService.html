AddNewCaseMessageNotification(CaseMessage)

Add a new notification for a new message.
If the message is from a Patient and the case has an assigned Doctor, the notification is given to the Doctor. If the message is from a Doctor, the notification is given to the Patient. Otherwise, both Patient and Doctor (if assigned) are given a notification.
